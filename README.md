# Student_Management_System
🎓 Student Management System

A simple and user-friendly Student Management System developed using HTML, CSS, JavaScript, PHP, and MySQL to manage student records efficiently.

✨ Features

* 🔐 Teacher Login
* 👨‍🎓 Add, Edit & Delete Students
* 📅 Attendance Management
* 📝 Marks Management
* 🚪 Secure Logout

🛠️ Technologies Used

* CSS
* HTML
* PHP
* MySQL
* XAMPP
